> Flowscripter Plugin API.

## Overview

This project provides a Plugin API for the Flowscripter system.

**This is a work in progress**


## Development

Firstly:

```
npm install
```

then:

Build: `npm run build`

Watch: `npm run watch`

Lint: `npm run lint`

Docs: `npm run docs`
